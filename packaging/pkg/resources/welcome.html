Mac Doctor

Version 2.1.0

Mac Doctor is a single-command diagnostic tool that tells you exactly why your Mac is slow.

This installer will:

  • Install mac-doctor to /usr/local/bin
  • Set up a weekly health scan with macOS notifications

What it checks:

  • CPU, memory, disk, thermals, battery health
  • Sleep blockers, security (FileVault, Firewall, SIP)
  • Background processes, Electron app overhead
  • WiFi quality, DNS speed, network diagnostics
  • AI tools & local model storage
  • Developer tools (Node, Docker, Xcode, Homebrew)
No data leaves your Mac. All diagnostics run locally using only built-in macOS tools. No network calls, no telemetry, no accounts. The source code is open and auditable on GitHub.
